## Color Usage Guidelines

### Primary Color Usage
- **Headers**: Use primary main color
- **Buttons**: Use primary main color
- **Links**: Use primary main color
- **Focus States**: Use primary main color
- **Active States**: Use primary dark color
- **Hover States**: Use primary light color

### Secondary Color Usage
- **Accents**: Use secondary main color
- **Highlights**: Use secondary light color
- **Badges**: Use secondary main color
- **Emphasis**: Use secondary main color
- **Hover**: Use secondary dark color

### Neutral Color Usage
- **Background**: Use background color
- **Cards**: Use paper color
- **Text**: Use text primary color
- **Secondary Text**: Use text secondary color
- **Dividers**: Use divider color

### Status Color Usage
- **Success**: #4CAF50 (Green)
- **Warning**: #FF9800 (Orange)
- **Error**: #F44336 (Red)
- **Info**: Organization Primary Color

## Implementation Notes

### Theme Configuration
- All colors are defined in `/frontend/src/config/theme.config.enhanced.js`
- Organization-specific palettes in `orgPalettes` object
- Component overrides in `componentOverrides` function
- Automatic theme switching based on organization

### Usage in Components
```jsx
import { useTheme } from '@mui/material/styles';

function MyComponent() {
  const theme = useTheme();
  
  return (
    <Box sx={{
      backgroundColor: theme.palette.primary.main,
      color: theme.palette.primary.contrastText,
    }}>
      Content
    </Box>
  );
}
```
