## Reconciliation Process

```
┌────────────────┐
│ Cron Schedule  │ Daily at 2 AM
│ (0 2 * * *)    │
└────┬───────────┘
     │ Trigger
     ▼
┌────────────────────────────────────────┐
│ Reconciliation Service                 │
│                                        │
│  ┌──────────────────────────────────┐ │
│  │ 1. Get All Exporters from CBC    │ │
│  └────────┬─────────────────────────┘ │
│           │                            │
│  ┌────────▼─────────────────────────┐ │
│  │ 2. For Each Exporter:            │ │
│  │    • Get from Fabric             │ │
│  │    • Compare Status              │ │
│  │    • Detect Mismatches           │ │
│  └────────┬─────────────────────────┘ │
│           │                            │
│  ┌────────▼─────────────────────────┐ │
│  │ 3. Repeat for:                   │ │
│  │    • Licenses                    │ │
│  │    • Certificates                │ │
│  │    • Shipments                   │ │
│  └────────┬─────────────────────────┘ │
│           │                            │
│  ┌────────▼─────────────────────────┐ │
│  │ 4. Handle Mismatches:            │ │
│  │    • Fabric Wins (state changes) │ │
│  │    • CBC Wins (detailed data)    │ │
│  │    • Manual Review (critical)    │ │
│  └────────┬─────────────────────────┘ │
│           │                            │
│  ┌────────▼─────────────────────────┐ │
│  │ 5. Record Results:               │ │
│  │    • reconciliation_log          │ │
│  │    • reconciliation_issues       │ │
│  └──────────────────────────────────┘ │
└────────────────────────────────────────┘
```

## Conflict Resolution Strategy

```
┌─────────────────────────────────────────┐
│ Mismatch Detected                       │
└────────┬────────────────────────────────┘
         │
         ▼
    ┌────────┐
    │ Type?  │
    └───┬────┘
        │
        ├─────────────────────────────────┐
        │                                 │
        ▼                                 ▼
┌───────────────┐                 ┌───────────────┐
│ State Change  │                 │ Detailed Data │
│ (status, etc) │                 │ (address, etc)│
└───────┬───────┘                 └───────┬───────┘
        │                                 │
        ▼                                 ▼
┌───────────────┐                 ┌───────────────┐
│ Fabric Wins   │                 │ CBC Wins      │
│ • Update CBC  │                 │ • Update      │
│   to match    │                 │   Fabric      │
│   Fabric      │                 │   metadata    │
└───────┬───────┘                 └───────┬───────┘
        │                                 │
        └─────────────┬───────────────────┘
                      │
                      ▼
              ┌───────────────┐
              │ Log Resolution│
              │ in sync_log   │
              └───────────────┘

┌─────────────────────────────────────────┐
│ Critical Conflict                       │
│ (payment, customs, etc)                 │
└────────┬────────────────────────────────┘
         │
         ▼
┌─────────────────────────────────────────┐
│ Manual Review Required                  │
│ • Insert into reconciliation_issues     │
│ • Alert admin                           │
│ • Wait for manual resolution            │
└─────────────────────────────────────────┘
```
